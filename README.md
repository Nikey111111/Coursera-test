# Coursera-test

This repository contains sample code.

## User Login

`login.py` provides a basic username/password login check using SHA-256 hashes.

Run the script:

```bash
python3 login.py
```

Default credentials:
- username: `admin`
- password: `password123`

## Tests

Run tests with `pytest`:

```bash
pip install pytest
python3 -m pytest
```
